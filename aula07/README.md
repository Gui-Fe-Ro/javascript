# Quarta aula

### Operadores

* aritméticos (+, -, *, /, %, **)
* atribuição (=, +=, -=, *=, /=, **=, %=, ++, --)
* relacionais (>, <, >=, <=, ==, !=, ===)
* lógicos (!, %%, ||)
* ternário (mádia>=7?"Aprovado":"Reprovado" -> Faço uma condição e os meus resultados)

### Ordem de precedência

* () -> ** -> * / % -> + -
